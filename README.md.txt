## Project Overview
This project is an interactive Python-based data tool built with Streamlit. It combines personal finance calculators and external financial data analysis to provide a user-friendly experience for financial decision-making.

## Features
- Loan Calculator: Calculate monthly loan repayments
- Savings Calculator: Estimate future value of personal savings
- Budget Analysis: Visualise monthly income and expenses with a pie chart
- Yahoo Finance Data Analysis: Download, clean, analyse, and visualise real stock data

## Dataset
- Source: Yahoo Finance
- Type: Historical daily stock prices
- Data process: Downloading → Cleaning → Analysis → Visualisation

## Python Libraries Used
streamlit
pandas
numpy
matplotlib
yfinance

## How to Run
1. Clone this GitHub repository
2. Install required packages: pip install -r requirements.txt
3. Run the app: streamlit run app.py
4. Access via browser: http://localhost:8501
