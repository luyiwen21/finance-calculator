# Personal Finance Calculator - ACC102 Track 4
A simple interactive personal finance tool built with Streamlit.

## 1. Problem & User
Help individuals quickly calculate loan payments, savings growth, and visualize monthly budget structure.

## 2. Data
User input data (no external dataset required).

## 3. Methods
- Python: streamlit, pandas, matplotlib
- Loan amortization formula
- Compound interest formula
- Pie chart for expense breakdown

## 4. Key Findings
- Clear monthly payment for loans
- Future value of long-term savings
- Visual spending structure to improve saving habits

## 5. How to run
1. Install dependencies: pip install -r requirements.txt
2. Run: streamlit run app.py
